# Light it Challenge

A mobile-first web app featuring interactive breathing and grounding exercises, with screen capture functionality.

## Features

- **Breathing Sphere** — Animated sphere guiding breathing exercises
- **Grounding Exercise** — Interactive grounding activity
- **Capture Tools** — Scripts to record grounding, senses, and video sessions

## Tech Stack

- HTML / JavaScript
- [Playwright](https://playwright.dev/) for capture automation

## Getting Started

Install dependencies:

```bash
npm install
```

## Project Structure

```
.
├── breathing_sphere.html   # Breathing exercise UI
├── grounding.html          # Grounding exercise UI
├── capture_grounding.js    # Capture script for grounding
├── capture_senses.js       # Capture script for senses
├── capture_video.js        # Video capture script
└── senses/                 # Senses exercise assets
```
